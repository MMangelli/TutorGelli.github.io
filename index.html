    • What is this website for?

    This is a personal website created to share files between friends and family members.

    How does it work?

    First, you will need to determine whether or not you are trying to upload a file or download a file.

    Uploading Files

    Select the "Upload" button below and then navigate on your device to the file you would like to share and select it.

    Downloading Files

    Select the file you would like on your device below and then click the "Download" button.

    Uploading

    This will be an Upload button.

    Downloading

    Select the desired file below.
